Nosso projeto

Nosso ideal é auxiliar as pessoas no dia-a-dia, automatizando processos para que seu cotidiano fique mais dinâmico e te dê oportunidade de otimizar e qualificar seu tempo!

Sobre equipe

O grupo foi formado durante o curso de Desenvolvimento WEB Fullstack .NET C# da Digital House, em parceria com Mercado Eletrônico através do projeto DiversiDev.

O curso teve duração de 180 horas, o que equivale a 3 meses de estudo com término em agosto, através de aulas 100% online e ao vivo, de segunda a sexta-feira.

O programa disponibilizou 30 bolsas, sendo 20 vagas de estágio e 10 bolsas para realizar o curso. Assim, o grupo é composto por três estagiários e uma bolsista.

No caso de quem foi selecionado e contratado como estagiário, foram privilegiados candidatos de grupos minoritários: mulheres, negros, pessoas trans e/ou com 50+ anos.

Ferramentas utilizadas

Para fazer o projeto, utilizamos o Microsoft Visual Studio para a parte do back-end, no qual desenvolvemos o API utilizando o swagger como conjunto de ferramenta para desenvolvimento.

No desenvolvimento do front-end, desenvolvemos através do Visual Studio Code, utilizando o Vue.Js, que é um framework JavaScript de código-aberto, focado no desenvolvimento de interfaces de usuário e aplicativos de página única.
